Category Parser#

Functions defined in this file parse CSV files with APP and URL categories to retrieve information about what categories should be managed, what categories should be allowed for all users, and what categories should be blocked outright.

lib.category_parser.parse_app_categories(filename)[source]#

Reads App categories from the input CSV file and builds a list of dictionaries with metadata.

Each element of the dictionary is a key that equals the column name, and a value that is a string in the cell.

Parameters:

filename (str) – Path to the CSV file containing app categories data.

Returns:

List of dictionaries containing app categories metadata (1 dictionary per category).

Returns None if the file is not found.

Return type:

list

Reads URL categories from the input CSV file and builds a list of dictionaries with metadata.

Parameters:

Returns:

List of dictionaries containing URL categories metadata (1 dictionary per category).

Returns None if the file is not found.

Return type:
